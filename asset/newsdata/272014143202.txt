<p style="margin: 0px 0px 1.5em; color: rgb(87, 94, 99); font-family: sans-serif; font-size: 14px; line-height: 22.399999618530273px;">มาตรฐานของระบบ User Interface ใหม่ของ Android มีชื่อเรียกว่า Meterial Design โดยได้แรงบันดาลใจมาจากกระดาษและหมึก ดูเรียบง่ายขึ้น สะอาดตามากขึ้น ถ้าใครใช้ Google+ หรือแอพของกูเกิลในช่วงหลังก็จะเห็นดีไซน์ในรูปแบบนี้มาอยู่แล้ว</p>
<p style="margin: 0px 0px 1.5em; color: rgb(87, 94, 99); font-family: sans-serif; font-size: 14px; line-height: 22.399999618530273px;">นอกจากนี้ยังมีระบบความลึก ซึ่งสามารถจัด Layer ต่างๆ ได้ ดูลึกแบบ 3 มิติมากขึ้น</p>

<p style="margin: 0px 0px 1.5em; color: rgb(87, 94, 99); font-family: sans-serif; font-size: 14px; line-height: 22.399999618530273px;">Android L จะมาพร้อมประสิทธิภาพที่รวดเร็วขึ้น รวมถึงรองรับระบบ 64-bit เพื่อรองรับแอพที่ใช้ Memory มากขึ้นในอนาคต ทำงานได้แบบ Cross Platform และไม่ต้องแก้ไขโค้ดเดิม</p>
<p style="margin: 0px 0px 1.5em; color: rgb(87, 94, 99); font-family: sans-serif; font-size: 14px; line-height: 22.399999618530273px;">ข่าวเก่าปี 2013 :&nbsp;<a href="http://www.macthai.com/2013/09/11/apple-introduce-iphone-5s-with-finger-scan/" sl-processed="1" style="outline: none; text-decoration: none; color: rgb(40, 142, 179);">แอปเปิลเปิดตัว iPhone 5s รองรับระบบ 64-bit ตัวแรกของโลก</a></p>
